# 電眼美女
## 游戲説明
這是2019年國立台北科技大學物件導向程式設計實習學期實作專案  
電眼美女是一款利用自己充滿魅力的電眼，將所有遇到的男生電暈變成自己粉
絲的系列遊戲。  
玩家將操作美女在學校搜尋目標使用電擊，同時必須避開訓導主任和其他
競爭對手，否則就無法成功迷惑對方。  
遊戲以特殊少見的題材、日常生活環境加上生動活潑的特效，創造出一款
具有獨特風味的遊戲，讓每個玩家都能享受當個萬人迷的滋味！
